pub struct Arena<const N: usize = 1024, const C: usize = 30> { /* private fields */ }
Expand description

N is the initial chunk size (in bytes) C is the maximum number of chunks chunks are allocated with exponential size, starting at N and doubles. the total size is N * (2^C - 1)
